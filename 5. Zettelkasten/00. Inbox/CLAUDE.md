# 00. Inbox 폴더

## 목적과 역할

**빠른 메모와 임시 캡처**를 위한 수집함입니다.
완성도보다 속도가 우선이며, 정기적으로 비워야 합니다.

## 어떤 노트가 들어가는가

- 즉흥적인 아이디어 메모
- 나중에 처리할 할 일 메모
- 아직 분류하지 않은 임시 노트
- 빠르게 캡처한 외부 자료

### 예시

- `20251013-독서중-떠오른-아이디어.md`
- `회의중-메모.md`
- `나중에-읽을-링크.md`

## 작업 방식과 규칙

1. **일단 저장**: 완성도 고민 없이 빠르게 기록
2. **주 1회 정리**: 매주 Inbox를 비우고 적절한 곳으로 이동
   - 가치 있는 내용 → `10. Literature` 또는 `20. Permanent`
   - 실행 항목 → `1. Projects` 또는 `2. Areas`
   - 불필요한 내용 → 삭제
3. **Inbox는 임시 공간**: 장기 보관 금지

### 정리 기준

| 내용 | 이동 위치 |
|------|-----------|
| 외부 자료 요약 | `10. Literature` |
| 내 생각/통찰 | `20. Permanent` |
| 할 일/프로젝트 | `1. Projects` |
| 참고 자료 | `3. Resources` |
| 불필요 | 삭제 |
