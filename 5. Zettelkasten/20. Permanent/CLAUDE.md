# 20. Permanent 폴더

## 목적과 역할

**내 언어로 재구성한 영구 지식 노트**를 보관하는 폴더입니다.
Zettelkasten의 핵심이며, 시간이 지나도 가치를 잃지 않는 원자적 지식 단위입니다.

## 어떤 노트가 들어가는가

- 하나의 아이디어/개념/주장을 담은 원자적 노트
- 내 언어로 완전히 재구성된 통찰
- 여러 출처를 종합한 나만의 관점

### 예시

- `습관은 정체성의 표현이다.md`
- `복리 효과는 지식에도 적용된다.md`
- `단순함이 창의성을 높인다.md`

## 작업 방식과 규칙

1. **원자성 원칙**: 노트 1개 = 아이디어 1개
2. **제목은 주장**: 명사구 또는 완전한 문장으로 작성
3. **최소 2개 연결**: 새 노트는 기존 노트 2개 이상과 `[[링크]]`
4. **내 언어로**: Literature 노트의 내용을 그대로 복사 금지
5. **출처 명시**: `source` 또는 `related`로 Literature 노트 연결

### Frontmatter 템플릿

```yaml
---
type: permanent
status: active
created: YYYY-MM-DD
updated: YYYY-MM-DD
source: "[[Literature 노트]]"
related: ["[[연관 노트1]]", "[[연관 노트2]]"]
tags: [permanent]
---
```

### 품질 기준

- [ ] 제목만 봐도 핵심 주장이 전달되는가?
- [ ] 내 언어로 작성되었는가?
- [ ] 최소 2개 노트와 연결되어 있는가?
- [ ] 6개월 후에도 의미가 있는가?
