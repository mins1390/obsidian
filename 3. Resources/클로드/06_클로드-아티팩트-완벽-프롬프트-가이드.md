# 클로드 아티팩트(Artifacts) 완벽 프롬프트 가이드

> 역프롬프팅 기법과 디테일 프롬프트 4선으로 코딩 없이 완성도 높은 앱·대시보드 즉석 제작

---

앱이나 웹 대시보드를 코딩 없이 눈앞에서 바로 만들어주는 '아티팩트' 기능, 제대로 된 결과물을 얻기 위한 프롬프트 설계법과 실전 템플릿을 공개합니다.

---

## 🍯 꿀팁: 역프롬프팅

### 아티팩트 프롬프트는 '클로드'에게 직접 짜게 하세요

비개발자가 "OO 앱 만들어줘"라고 단순하게 명령하면 허술한 결과물이 나옵니다. 가장 좋은 방법은 **내가 원하는 기획을 먼저 말하고, 클로드에게 완벽한 프롬프트를 다시 뽑아달라고 지시**하는 것입니다.

**[역프롬프트 예시 — 그대로 복사해서 쓰세요]**

> "나는 현재 OO을 위한 웹 애플리케이션을 만들고 싶어. 주요 기능으로 A, B, C가 반드시 포함되어야 해.
> 네가 프론트엔드 개발자이자 UX/UI 디자이너라고 생각하고, 클로드 아티팩트를 통해 한 번에 완성도 높은 결과물을 구현할 수 있도록 가장 구체적이고 완벽한 '프롬프트'를 나 대신 작성해 줘."

클로드가 만들어준 구체적인 프롬프트를 그대로 새 채팅창에 복사해 넣으면, 첫 시도부터 퀄리티 높은 앱이 완성됩니다.

---

## 📏 아티팩트 프롬프트 4대 원칙

직접 프롬프트를 작성할 때는 다음 4가지를 반드시 포함하세요.

1. **구체적인 기능 명시:** (예: "저장 버튼, 텍스트 입력 창, 기간 필터링 드롭다운을 넣어줘")
2. **명확한 레이아웃 지정:** (예: "메뉴는 왼쪽 사이드바에, 핵심 차트는 화면 중앙 상단에 배치해 줘")
3. **결과물 포맷 지정:**
   - 동작하는 앱/웹 화면이 필요할 때 → `"React(Tailwind CSS)로 만들어 줘"`
   - 깨지지 않는 다이어그램이나 구조도가 필요할 때 → `"SVG 포맷으로 그려 줘"`
4. **점진적 수정:** 한 번에 완벽할 수는 없습니다. 화면을 띄워놓고 "버튼 색상을 파란색으로 바꿔줘", "목록 폰트를 더 크게 해줘"처럼 대화하듯 수정하세요.

## 🎨 아티팩트 실전 디테일 프롬프트 4선

### 1️⃣ 궁극의 데일리 습관 트래커

```
습관 트래커 앱을 React 아티팩트로 만들어줘.

[기본 구조 및 디자인]
- 다크 모드 적용 (배경 #0f0f0f, 카드 #1a1a1a)
- 최대 너비 480px로 모바일 환경에 최적화
- 하단 탭 네비게이션: 체크 / 통계 / 설정
- 카드형 레이아웃, border-radius 14px 적용
- 부드러운 화면 전환(transition) 전체 적용

[상단 헤더 - 고정 영역]
- 날짜 네비게이션 (◂ 오늘 ▸) 구성, 미래 날짜 이동 불가
- 원형 프로그레스 게이지 (달성률 %, 색상: 20% 미만 회색 → 50% 파랑 → 80% 초록)
- 연속 기록 🔥 카운터 (매일 70% 이상 달성 시 +1)
- 프리즈 ❄️ 잔여 횟수 (월 2회, 연속 기록 보호용)

[습관 항목 - 시간대별 그룹핑]
- ☀️ 아침 루틴 1. 기상 시간 (시간 입력란): 목표 07:00 이전이면 초록 뱃지, 이후면 빨간 뱃지
- ☀️ 아침 루틴 2. 물 마시기: 💧 아이콘 10개 가로 배열, 탭하면 채워짐 (1잔=250ml)
- ☀️ 아침 루틴 3. 영양제: 토글 스위치
- 🌤 오후 루틴 1. 운동: 토글 ON 시 종류 태그 선택 + 시간(분) 입력창 활성화
- 🌤 오후 루틴 2. 독서: 토글 ON 시 페이지 수 입력창 활성화, 이번 달 누적 페이지 표시
- 🌙 저녁 루틴 1. 취침 전 폰 끄기: 토글 스위치
- 🌙 저녁 루틴 2. 오늘 하루 기분: 기분 이모지 5개(😫😐🙂😊🤩) 중 택 1
- 📌 내 습관 (커스텀): "+" 버튼으로 최대 3개 추가 가능, ✕ 버튼으로 삭제

[게이미피케이션 요소]
- 하루 100% 달성 시 화면 전체에 컨페티(종이꽃가루) 애니메이션 실행
- 과거 날짜에 달성률 70% 미달 시 프리즈 사용 버튼 활성화

[통계 및 설정 탭]
- 이번 주 히트맵: 월~일 구성, 색 진하기로 달성률 표시
- 이번 달 캘린더 그리드: 날짜별 색상 표시
- 요약 카드: 연속 기록, 이번 달 평균 %, 기록한 날수 명시
- 데이터 내보내기 (JSON 다운로드) 및 전체 초기화 버튼

[데이터 저장]
- 모든 데이터는 localStorage에 저장하여 새로고침해도 유지
- 날짜별 키(YYYY-MM-DD)를 사용하여 데이터 관리
```

### 2️⃣ 헬스장 필수, 운동용 인터벌 타이머

```
운동용 인터벌 타이머를 React 아티팩트로 만들어줘.

[설정 화면 - 시작 전]
- 상단 프리셋 카드 3개 가로 배열
- 프리셋 1. Tabata (운동 20초 / 휴식 10초 / 8라운드)
- 프리셋 2. HIIT (운동 30초 / 휴식 30초 / 10라운드)
- 프리셋 3. Custom (직접 설정)
- 중앙 설정 영역: 운동 시간, 휴식 시간, 라운드 수를 +/- 버튼으로 조절 (5초 단위)
- 하단: 큰 "START" 버튼과 총 운동 시간 미리보기 제공

[타이머 화면 - 실행 중]
- 상단 텍스트: "Round 3 / 8" 및 전체 남은 시간 작게 표시
- 중앙: 큰 원형 SVG 프로그레스 링 (시간에 따라 부드럽게 줄어드는 애니메이션)
- 링 내부: 72px 이상의 대형 카운트다운 숫자 (고정폭 폰트 적용)
- 링 하단: 현재 상태 라벨 ("🏋️ 운동" 또는 "😮‍💨 휴식")
- 라운드 진행 인디케이터: ● ● ● ○ ○ ○ ○ ○
- 최하단 조작부: 일시정지 ⏸️ / 정지 ⏹️ / 스킵 ⏭️ 버튼

[상태별 색상 전환]
- 준비 단계 (3초 카운트다운): 배경 앰버 톤, 링 #FFD600
- 운동 단계: 배경 어두운 빨강 톤, 링 #FF4444
- 휴식 단계: 배경 어두운 초록 톤, 링 #00E676
- 완료 단계: 배경 보라 톤, "완료!" + 컨페티 애니메이션
- 마지막 3초: 숫자가 커졌다가 작아지는 펄스 애니메이션

[효과음 사운드]
- Web Audio API (OscillatorNode) 활용 (외부 파일 없이)
- 상태 시작 시: 짧은 비프 2회 (440Hz, 100ms)
- 마지막 3초: 매초 비프 1회 (880Hz, 80ms)
- 운동 전체 완료 시: 긴 비프 1회 (660Hz, 500ms)
```

### 3️⃣ 직장인 필수, 점심 메뉴 랜덤 룰렛

```
"오늘 뭐 먹지?" 랜덤 메뉴 룰렛을 React 아티팩트로 만들어줘.

[상단 및 하단 구조]
- 상단 카테고리 탭: 전체 / 한식 / 중식 / 일식 / 양식 / 분식 / 아시안
- 하단 좌측: "싫어! 다시 돌려" 버튼 (뽑힌 메뉴 제외 후 재회전)
- 하단 우측: "메뉴 편집" 버튼

[중앙 룰렛 휠 영역]
- 화면 너비의 85% 크기로 Canvas 2D 원형 룰렛 구현
- 색상 테마: 밝은 파스텔 팔레트 순환
- 상단 12시 방향에 빨간색 삼각형 포인터 배치
- 휠 정중앙에 "SPIN 🎰" 버튼

[기본 메뉴 데이터베이스]
- 한식: 김치찌개, 된장찌개, 순두부찌개, 제육볶음, 비빔밥, 삼겹살, 국밥, 돈까스 등 15종
- 중식: 짜장면, 짬뽕, 탕수육, 마라탕, 마라샹궈, 양꼬치
- 일식: 초밥, 라멘, 우동, 돈카츠, 연어덮밥, 카레
- 양식: 파스타, 피자, 햄버거, 스테이크, 샐러드, 샌드위치
- 분식: 떡볶이, 순대, 튀김, 라면, 김밥, 만두
- 아시안: 쌀국수, 팟타이, 카오팟, 샤브샤브, 똠양꿍

[핵심 스핀 애니메이션]
- requestAnimationFrame으로 총 4초간 부드럽게 회전
- 최소 5바퀴(1800도) 이상 회전 후 랜덤 목표 각도에서 정지
- easing: 초반 매우 빠르고 후반 아주 천천히 감속
- 회전 중 세그먼트 경계선 지날 때마다 Web Audio API로 틱 사운드 재생

[결과 표시 모달]
- 뽑힌 메뉴 카테고리에 맞는 큰 이모지 표시
- 컨페티 애니메이션 실행
- 모달 내 버튼: "좋아! 🎉" (닫기), "싫어! 🔄" (메뉴 제거 후 재스핀)
- 커스텀 메뉴 변경사항 localStorage에 저장
```

### 4️⃣ 업무 효율 극대화, 뽀모도로 타이머 & 칸반 보드

```
뽀모도로 타이머가 결합된 태스크 칸반 보드를 React 아티팩트로 만들어줘.

[전체 레이아웃]
- 화면을 좌우 6:4 비율로 분할 (좌측: 칸반 보드, 우측: 타이머 패널)
- 전체 다크 테마 (배경 #121212, 패널 #1E1E1E)
- 모바일 접속 시 상단 타이머, 하단 칸반 보드로 세로 스택형 전환

[좌측: 인터랙티브 칸반 보드]
- 3개 상태 칼럼: "할 일", "진행 중", "완료"
- 태스크 카드 드래그 앤 드롭으로 칼럼 간 이동 가능
- 태스크 카드 상단에 우선순위 태그 (높음/보통/낮음) 색상별 표시
- 새 태스크 입력 폼 + ➕ 버튼
- 태스크 클릭 시 제목 수정 및 삭제 가능

[우측: 뽀모도로 집중 타이머]
- 모드 전환 탭: 🍅 집중(25분) / ☕ 짧은 휴식(5분) / 🛋️ 긴 휴식(15분)
- 중앙: 원형 타이머 (남은 시간에 따라 링 테두리 줄어드는 애니메이션)
- 타이머 시작 / 일시정지 / 초기화 버튼
- 오늘 완료한 뽀모도로 세션 횟수를 🍅 아이콘으로 표시

[상태별 시각/청각 효과]
- 집중 모드: 링 색상 #FF6B6B, 두근거리는 펄스 이펙트
- 휴식 모드: 링 색상 #4ECDC4
- 시간 종료 시: 화면 전체 보더라인 반짝임 + Web Audio API 종소리 1회

[데이터 저장]
- 칸반 카드 위치 및 완료된 뽀모도로 횟수 localStorage에 즉시 동기화
```
