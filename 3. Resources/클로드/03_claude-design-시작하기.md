# Claude Design 시작하기 — 프롬프트 한 줄로 만드는 진짜 디자인

> 코드 없이 프롬프트 한 줄로 완성된 웹 디자인을 만들어주는 Claude Design 모드 입문 가이드

---

## Claude Design이 뭔가요?

Claude가 웹에서 바로 제공하는 **디자인 전용 모드**. 브랜드 톤을 이해하고, 레퍼런스를 보고, 실제 페이지로 만들어준다.

| 특징 | 설명 |
|------|------|
| 코드 없이 | 프롬프트 → 완성된 페이지 |
| 브랜드 학습 | 이미지·톤·컬러 반영 |
| 웹에서 바로 | 설치·플러그인 불필요 |
| 실전 퀄리티 | 바로 공유 가능한 결과물 |

> 핵심 → 디자이너가 아니어도 괜찮다. 원하는 느낌만 설명하면 된다.

---

## STEP 01 · 시작하기

브라우저에서 **claude.ai/design** 접속. 로그인만 되어 있으면 즉시 사용 가능.

> ⚠️ 데스크탑 앱에서는 지원되지 않음.

---

## STEP 02 · Design System 설정

이름·브랜드 컬러·타입만 정하면 Claude가 톤을 기억한다.

- **System Name**: 내 브랜드명
- **Brand Color**: 색상 코드 입력 (예: `#C6A26B`)
- **Typography**: 원하는 폰트 (예: Noto Serif KR · Pretendard)
- **Reference Images**: 레퍼런스 이미지 첨부

> Tip. 시스템을 한 번 만들어 두면 다음 작업에도 계속 재사용 가능.

---

## STEP 03 · 레퍼런스

말로 설명하기 어려운 **무드·레이아웃·컬러감**은 이미지로 전달. 2~4장이 최적 (너무 많으면 결과가 흐려짐).

---

## STEP 04 · 프롬프트

원하는 느낌, 레이아웃, 분위기를 구체적으로 작성.

예시: `"따뜻한 톤의 슬로우 패션 브랜드 랜딩 페이지"`

결과물: 타입 / 컬러 / 레이아웃 / 카피 모두 포함.

---

## 만들 수 있는 것들

| 종류 | 설명 |
|------|------|
| 랜딩 페이지 | 브랜드·제품·SaaS |
| 대시보드 | 차트·테이블·관리자 |
| 앱 온보딩 | iOS·Android |
| 이메일·뉴스레터 | 레이아웃 템플릿 |
| 포트폴리오 | 개인·스튜디오 |
| 카드뉴스 | 시리즈 |

---

## 주의사항

> ‼️ 카드뉴스 한 세트를 만들면 일주일치 크레딧이 소진될 수 있다. 설정 → 사용량에서 확인 가능. 일주일 후 리셋.
