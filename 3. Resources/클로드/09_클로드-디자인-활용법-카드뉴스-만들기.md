# [가이드] 클로드 디자인 활용법 1탄 : 카드뉴스 만들기편

> Claude Design의 4가지 컨텍스트 설정과 2가지 방법(기존 디자인 활용·신규 시안 제작)으로 카드뉴스 즉시 완성

---

이 가이드는 Anthropic이 만든 디자인 전용 AI 툴 **Claude Design**으로 카드뉴스를 만드는 방법을 담은 실전 가이드입니다.

---

## 1️⃣ Claude Design이란?

- [Claude Design 바로가기](https://claude.ai)

클로드 디자인은 Anthropic Labs에서 만든 리서치 프리뷰 디자인 툴입니다. 상단 탭 4개 중 목적에 맞는 걸 골라서 시작하면 됩니다.

| 탭 | 설명 |
|----|------|
| **Prototype** | 실제 클릭·화면전환이 되는 인터랙티브 앱/웹 결과물 |
| **Slide deck** | HTML 기반 슬라이드. 디자인 시스템 적용 시 일관된 스타일 |
| **From template** | 완성된 틀에 내용만 채워 빠르게 시작하는 방식 |
| **Other** | 카드뉴스, 배너, 포스터 등 자유 형식 디자인 |

---

## 2️⃣ 프로젝트 생성 시 컨텍스트 설정

### 1. Design System
이미 학습시킨 디자인으로 계속 동일한 스타일 생성 가능. 브랜드 컬러, 폰트, 컴포넌트 규칙 등을 Claude에게 등록해두는 개념. 한 번 만들어두면 이후엔 내용만 넣어도 동일한 스타일로 결과물이 나옵니다.

### 2. Add screenshot
따라하고 싶은 디자인 캡처해서 업로드. PPT, 카드뉴스, 레퍼런스 디자인 등을 캡처해서 업로드하면 Claude가 스타일을 분석합니다.

### 3. Attach codebase
기존 프론트엔드 코드 폴더를 통째로 첨부. 대규모 코드베이스도 폴더째로 드래그 앤 드롭 가능합니다.

### 4. Drag in a Figma file
피그마 파일 첨부해서 그 디자인 그대로 따라하기. 피그마 로고 클릭 → File → Save local copy → fig 포맷으로 저장 후 첨부하면 됩니다.

---

## 3️⃣ 실전 활용 — 카드뉴스 제작하기

### 방법 A — 기존 디자인이 있는 경우

#### [Step 1] 디자인 참고자료 첨부
- **Add screenshot**: 따라하고 싶은 카드뉴스 디자인 캡처 후 업로드
- **Drag in a Figma file**: 피그마에서 작업했다면 카드뉴스 피그마 파일 첨부

#### [Step 2] 상황에 따라 방법 선택

**Case A — 내 카드뉴스 디자인인 경우**
레이아웃 구조가 동일하다면, 내용만 첨부하고 "내용만 교체해줘"라고 하면 끝!

**Case B — 내 카드뉴스 디자인이 아닌 경우**
레이아웃 구조가 다를 수 있으니, 먼저 스크립트로 변환하는 작업부터 시작합니다.

```
"해당 카드뉴스 디자인에 내용을 삽입할거야. 내용이 알맞게 들어갈 수 있도록 내용을 정리하고 삽입해줘"
```

---

### 방법 B — 나만의 고유한 디자인 새로 제작

#### [Step 1] 디자인 시안 제작 (프롬프트 입력)

`Other` 프로젝트 생성 후 채팅창에 아래 프롬프트를 입력합니다. **컬러 팔레트의 컬러 코드만 원하는 색으로 변경**하세요. 컬러 코드는 [컬러헌트](https://colorhunt.co/)에서 찾을 수 있습니다.

```
# 역할: 너는 UI/UX 디자이너이자 인스타그램 콘텐츠 전략가야.
아래의 브랜드 가이드를 바탕으로 '카드뉴스 디자인 템플릿' 3가지 시안을 제작해줘.

## 컬러 팔레트(필수 준수):
- 7B61FF, F8F9FD, EBE9FC, 5B4ED6

## 디자인 원칙:
- 본문은 가독성이 좋은 Sans-serif 서체를 사용해줘.
- 텍스트는 주로 좌측 정렬을 사용해줘.
- 2026년 트렌드 감성이 느껴지도록 깔끔하게 구성해줘.
- 카드나 버튼에 배경색과 비슷한 톤의 아주 연한 그림자를 적용해서 입체감을 살려줘.
- 중요한 숫자나 키워드만 색으로 강조해서 시선이 분산되지 않도록 조절해줘.
- 슬라이드 한페이지의 사이즈는 1080 x 1350이야.

## 제작할 카드 구성:
- 표지: 좌측 하단에 큰 제목
- 본문: 다양한 구조의 레이아웃 3페이지
- 마무리 페이지: 중앙에 CTA 문구, 하단에 인스타그램 프로필 정보
```

#### [Step 2] 시안 선택 및 템플릿 저장

생성된 시안 중 마음에 드는 것을 골라 템플릿으로 저장 요청합니다.

```
"시안 A의 html 코드만 템플릿으로 생성하고 저장해줘."
```

#### [Step 3] 템플릿으로 프로젝트 생성

클로드 디자인이 작성된 코드를 기반으로 전용 템플릿을 빌드합니다.

1. 채팅창에 생성된 **'템플릿 열기'** 버튼을 클릭
2. 하단의 **`Published`** 버튼을 켜서 공유 가능한 상태로 만들기
3. **`New from this template`** 클릭하여 새 프로젝트 생성

이제 디자인은 유지된 채, 해당 프로젝트 내 채팅창에 **내용만 입력**하면 카드뉴스가 즉시 완성되는 워크플로우가 세팅됩니다.

#### [Step 4] 수정 및 다운로드

1. **편집 모드 진입**: 결과물 상단의 **[Edit]** 버튼 클릭
2. **요소 수정**: 텍스트 문구·폰트·사이즈·색상 변경, 블록 위치 이동 및 크기 조절
3. **PNG로 저장**: 채팅창에 아래 입력

```
"png로 변환해서 다운로드할 수 있게 만들어줘"
```
