# 혼자 일하는데 팀이 있는 것처럼 일하는 법

> Claude, MCP, Make.com, OpenRouter 등 AI 도구를 조합해 1인으로 팀 수준의 생산성을 내는 리소스 모음

---

Claude와 여러 AI 도구를 팀원처럼 활용하는 링크 모음. 아래 도구들을 조합하는 워크플로우에 집중합니다.

- **Claude Code** — 코드 작성·자동화 전담
- **MCP (Model Context Protocol)** — 외부 서비스 연동
- **Make.com** — 워크플로우 자동화
- **OpenRouter** — 멀티 모델 라우팅
- **Notion** — 지식 관리·컨텍스트 저장
