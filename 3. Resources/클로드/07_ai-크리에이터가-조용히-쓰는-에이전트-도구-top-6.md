# AI 크리에이터가 조용히 쓰는 에이전트 도구 TOP 6

> Claude Skills 생태계에서 주목할 오픈소스 에이전트 도구 6가지 — GitHub 링크와 핵심 기능 요약

---

## 1. stop-slop

AI 티나는 문체(throat-clearing, 과장 어휘, em dash 등)를 자동 제거하는 Skill

- GitHub: https://github.com/hardikpandya/stop-slop

---

## 2. last30days-skill

Reddit, X, YouTube, HN, Polymarket, TikTok, Instagram, Bluesky 등 **최근 30일 콘텐츠를 종합 리서치**해서 근거 있는 요약 + 복사-붙여넣기 프롬프트를 생성

- GitHub: https://github.com/mvanhorn/last30days-skill

---

## 3. excalidraw-diagram-skill

자연어 → Excalidraw 다이어그램 자동 생성. Playwright 기반 렌더 검증 루프 내장 (본인 아웃풋 보고 스스로 레이아웃 수정)

- 관련 링크: https://mlflow.org/blog/evaluating-skills-mlflow

---

## 4. duckdb-skills

DuckDB 기반 로컬 SQL/데이터 분석 Skill

- GitHub: https://github.com/duckdb/duckdb-skills

---

## 5. browser-use

AI에게 브라우저 조작 능력을 부여하는 파이썬 오픈소스 프레임워크

- GitHub: https://github.com/browser-use/browser-use

---

## 6. Valyu agent-skills

arXiv, PubMed, SEC, FRED, Polymarket, 뉴스 등 **40M+ 전문 데이터 소스**를 하나의 Skill로 코딩 에이전트에 연결

- Docs: https://docs.valyu.ai/integrations/agent-skills
